# Price Scraper Prompts

Run an individual series with:

```bash
node --import tsx/esm scripts/scrapePricing.ts --series="SERIES NAME"
```

You can also use the npm script form:

```bash
npm run scrape:pricing -- --series="SERIES NAME"
```
